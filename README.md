# Android 서비스 테스트

이 프로젝트는 다양한 유형의 Android 서비스를 구현하는 방법을 보여줍니다.

## 기능

*   **포그라운드 서비스**: `MyForegroundService`
*   **백그라운드 서비스 (WorkManager 사용)**: `MyBackgroundService`
*   **바운드 서비스**: `MyBoundService`

## 사용 방법

1.  저장소를 복제합니다.
2.  Android Studio에서 프로젝트를 엽니다.
3.  Android 기기 또는 에뮬레이터에서 앱을 빌드하고 실행합니다.
4.  화면의 버튼을 사용하여 서비스를 시작/중지하고 바운드 서비스에서 데이터를 가져옵니다.

## 서비스

### MyForegroundService

영구 알림을 표시하는 포그라운드 서비스입니다.

*   **서비스 시작**: 포그라운드 서비스를 시작합니다.
*   **서비스 중지**: 포그라운드 서비스를 중지합니다.

### MyBackgroundService

지연 가능한 백그라운드 작업을 위해 `WorkManager`를 사용하여 구현된 백그라운드 서비스입니다.

*   **백그라운드 서비스 시작**: 일회성 작업 요청을 큐에 추가합니다.

### MyBoundService

구성 요소가 바인딩하고 상호 작용할 수 있도록 허용하는 바운드 서비스입니다.

*   **서비스 바인드**: 서비스에 바인딩합니다.
*   **서비스 언바인드**: 서비스에서 언바인딩합니다.
*   **난수 가져오기**: 서비스의 메서드를 호출하여 난수를 가져옵니다.
